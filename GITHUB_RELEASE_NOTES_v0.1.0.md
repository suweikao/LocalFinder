# LocalFinder v0.1.0

首个 macOS MVP 发布版。

## 安装包

- `LocalFinder_0.1.0_aarch64.dmg`

## 更新内容

- 支持 macOS Apple Silicon 安装包。
- 打开应用后自动启动本地后端服务。
- 支持添加索引目录并自动开始建立索引。
- 支持本地全文检索与文件名检索。
- 搜索结果展示命中文本、命中行次、文档匹配数。
- 右侧展示文档全文并高亮关键词。
- 支持浅色 / 深色主题切换。
- 支持中文 / 英文界面切换。
- 使用 LocalFinder 品牌图标作为应用与安装包图标。

## 隐私说明

LocalFinder 是本地优先软件。文件内容和索引默认只保存在用户本机，不上传文档。
