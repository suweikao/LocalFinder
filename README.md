# LocalFinder

LocalFinder 是一款本地全文索引与搜索桌面工具。它会在用户本机扫描指定目录，抽取文档文本并建立本地全文索引，所有索引数据默认保存在本地。

## 下载

请在 GitHub Releases 中下载对应平台安装包。

当前版本：

- macOS Apple Silicon: `LocalFinder_0.1.1_aarch64.dmg`

Intel Mac 需要单独的 x64 构建环境；Windows 和 Linux 安装包需要在对应系统构建后发布。

## 隐私

- 文件内容默认仅保存在用户本机。
- 不上传用户文档。
- 不调用云端 OCR 或云端模型。
- 索引目录由用户选择，可在应用内移除。

## 功能

- 本地目录扫描
- 文档全文索引
- 全文 / 文件名检索
- 搜索命中行展示
- 右侧文档全文预览
- 点击结果跳转到命中行
- 本地后端随桌面应用自动启动
- 选择索引目录后自动建立索引
