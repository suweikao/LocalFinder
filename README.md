# LocalFinder

LocalFinder 是一款本地全文索引与搜索桌面工具。它会在用户本机扫描指定目录，抽取文档文本并建立本地全文索引，所有索引数据默认保存在本地。

## 下载

请在 GitHub Releases 中下载对应平台安装包。

当前版本：

- macOS Apple Silicon: `LocalFinder_0.1.3_aarch64.dmg`

Intel Mac 需要单独的 x64 构建环境；Windows 和 Linux 安装包需要在对应系统构建后发布。

## 隐私

- 文件内容默认仅保存在用户本机。
- 不上传用户文档。
- 不调用云端 OCR 或云端模型。
- 索引目录由用户选择，可在应用内移除。
- 索引数据库可由用户选择本地存放路径。

## 功能

- 本地目录扫描
- 文档全文索引
- 全文 / 文件名检索
- 搜索命中行展示
- 右侧文档全文预览
- 点击结果跳转到命中行
- 本地后端随桌面应用自动启动
- 已建立索引持久保存，启动后直接复用
- 用户明确选择目录后再建立索引
- 可选择索引数据库本地存放位置
- 左侧栏区分索引范围和索引数据库，避免目录来源与索引存放位置混淆
- 限定检索采用类似 Everything 的分类筛选：文档、表格、演示、网页、数据、代码
- 移除索引目录时同步清理该目录下的索引数据
